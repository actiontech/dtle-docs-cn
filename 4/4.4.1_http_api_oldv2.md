## 创建/更新 一个作业

API: `POST /v1/jobs`

样例: **job1.json的内容说明参看[作业(job)配置](./4.3_job_configuration.md)**

```
> curl -H "Accept:application/json" -XPOST "172.17.5.6:8190/v1/jobs" -d @job1.json
```

## 删除一个作业

API: `DELETE /v1/job/{ID}`

样例: 

```
> curl -H "Accept:application/json" -XDELETE "172.17.5.6:8190/v1/job/8ce4b408-8c64-41b9-04d7-baf451348e89"
```

## 暂停一个作业 和 继续一个作业
由于nomad plugin API限制，无法实现。需要通过删除来暂停job，重新创建来继续job。见[HTTP API](./4.4_http_api.md)
